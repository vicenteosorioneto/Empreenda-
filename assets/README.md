# Assets

Esta pasta contém os recursos do aplicativo como ícones e imagens.

## Arquivos necessários:

- `icon.png` - Ícone do app (1024x1024px)
- `splash.png` - Tela de splash (1242x2436px)
- `adaptive-icon.png` - Ícone adaptativo Android (1024x1024px)
- `favicon.png` - Favicon para web (16x16px ou 32x32px)

## Como adicionar:

1. Substitua os arquivos desta pasta pelas suas imagens
2. Mantenha os nomes dos arquivos
3. Respeite as dimensões recomendadas

## Ferramentas úteis:

- [Expo Icon Generator](https://expo.github.io/tools/)
- [App Icon Generator](https://appicon.co/)
- [Figma](https://figma.com) para criar designs
